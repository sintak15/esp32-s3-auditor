# Contributing

Thanks for taking the time to contribute.

## Ground rules
- This project is intended for **authorized** testing and education. Please keep discussions and contributions aligned with that goal.
- Keep PRs focused and small when possible.
- Do not commit build outputs (`build/`, `*.bin`, `*.elf`, `*.map`) or local IDE state.

## Reporting bugs
When filing an issue, include:
- Hardware details (board + display)
- ESP32 Arduino core version
- Arduino IDE / `arduino-cli` version
- Steps to reproduce
- Serial logs (if available)

## Submitting PRs
- Ensure it compiles.
- If the change affects runtime behavior, smoke-test on hardware.
- Prefer clear names and consistent formatting (match surrounding code).
